# Blinkit-Sales-Analysis-Dashboard

## Overview

This project presents a comprehensive Power BI dashboard designed to analyze sales performance on the Blinkit platform. The dashboard provides actionable insights into customer behavior, product category performance, outlet characteristics, and overall revenue trends.

## Key Objectives

- Understand sales distribution across product categories and outlet types.
- Identify patterns in item visibility, pricing, and outlet location.
- Evaluate the impact of outlet establishment year and fat content on sales.
- Support business decisions using interactive visual analytics.

## Dataset Description

The dataset includes the following major fields:

- Item Identifier
- Item Weight
- Item Fat Content
- Item Visibility
- Item Type
- Item MRP
- Outlet Identifier
- Outlet Establishment Year
- Outlet Size
- Outlet Location Type
- Outlet Type
- Item Outlet Sales

## Dashboard Components

- KPI Cards: Total sales, average MRP, average item weight, and total outlets.

Bar Charts: Sales by item type and outlet type.

Pie/Donut Charts: Sales distribution by item fat content.

Line Chart: Sales trend across outlet establishment years.

Scatter Plot: Relationship between item visibility and item outlet sales.

Maps (if applicable): Regional breakdown of sales by outlet location.

## Technologies Used

Power BI Desktop for data modeling and visualization.

DAX Measures to calculate sales KPIs and relationships.

Power Query for data cleaning and transformation.

## Insights Delivered

High-performing product categories and outlet types.

The effect of item characteristics (weight, fat content, visibility) on sales.

Influence of outlet size and location on performance.

Pricing strategies supported by MRP and sales correlation.
